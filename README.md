# Trabajo 1: Sintaxis Python

Objetivo:
Desarrollar una calculadora de promedios escolares en Python utilizando variables, operadores, estructuras de control y funciones básicas.

Requisitos mínimos:
- Implementar una función `calcular_promedio(notas: list) -> float`.
- Programa interactivo `main.py` que pida al usuario el número de notas y las notas, y muestre: promedio (dos decimales), nota mínima, nota máxima y una clasificación (Aprobado si promedio >= 5.0).
- Añadir pruebas con pytest en `tests/test_promedio.py`.

Entrega:
- Archivos en la raíz: `main.py`, `promedio.py`, `requirements.txt` (si aplica) y `tests/`.
